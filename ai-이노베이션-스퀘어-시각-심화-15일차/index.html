신경망 리뷰

Y가 [0.6, 0.1, 0.3]이라는 건
정답의 차원도 3차원으로 동일하다는 의미이다.
loss가 T-Y의 평균이고 Y는 행렬이니까.
=> T : [1, 0, 0], [0, 1, 0], [0, 0, 1]

=> sparse : 1, 2, 3
cf. 다하분류에서는 정답이 one hot encoding으로 표현된다.
spase라는 건 명시적으로 표현하지 않을 뿐, 내부적으로 해준다.

loss를 계산할 때 각 항은 독립적이다.
이미지가 자동차든, 트럭이든, 비행기이든 object가 존재하는 위치 값 (x,y)는 독립적인 값이다.
=> yolo의 loss function만 봐도 알 수 있다.

INSIGHT

loss 값을 계산하기 위해서는 정답 T와 신경망의 출력값 Y 차원이 동일해야 한다.

Object Detection

cf. 피드 포워드 : 신경망으로 데이터를 흐르게 하는 거

차원이 더 늘어난다.

YOLO (You Only Look Once)

one-shot detection 방식 (2015년)

cf. fps = frame per second

학습한 것에 이미지 몇 천 번씩 넣고 쪼개서 분석하는 게 R-CNN,
이미지를 한 번 넣어서 다 분석하는 게 YOLO이다.
YOLO는 transfer learning의 기본 구조이다.

cf. 2012 CNN -> 2014 Transfer Learning -> 2015 YOLO

YOLO v1
params : 7x7x(2x5+20)=1470

하나의 grid에 대해 2개의 bounding box가 있다. bounding box 크기는 알 수 없다.
모든 bounding box에는 confidence score가 있다.

용어 정리
  • grid cell

grid를 n by n으로 분할한 하나의 cell

  • Pr(object)

grid cell에 물체가 있을 확률
=> 물체가 bbox에 안 걸리면 학습 x
=> object를 포함하도록 bbox 위치를 조정하는 것 = 학습
=> bbox의 크기가 object에 근사하게 작아지면 IoU가 커진다.

  • ground truth

  • Pr(Class|Object)

  • IoU

  • confidence score

  • class specific confidence score

하나의 grid cell에 있는 2 bbox에 대하여
class에 대한 확률을 얼마나 신뢰할 수 있는지?
=> 구체적인 object를 찾을 확률. 20개 중 가장 가능성이 큰 게 무엇인지?

  • NMS

bbox 여럿 중에서 가장 확률 높은 걸 남기는 것
=> bbox끼리 많이 겹쳐져 있다면, 가장 많이 겹쳐진 것만 남긴다.
