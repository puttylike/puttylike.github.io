  • numpy 쓸 땐 for 쓸 일이 거진 없다. cf. 함수 return 시 type을 썼는데, numpy의 형상을 알기 위해 .shape 을 자주 쓰게 된다.

​(3) 변형 : A.reshape()
vector를 matrix로 바꾸거나 matrix를 다른 형상의 matrix로 변경할 때 사용한다.

cf. 가상화 virtulization 10G 5명 나눠쓰기 4G (per 1인)

  • 인당 2G 이면 user preference
  • 누군가 4G를 쓰면 나머지 6G를 나눠 써야 함
    => 물리적인 기준 10G를 넘어갈 수 없음

​ex. A.shape : (3,4) 일 때

  • reshape(-1, 3) => -1이 나오면 행은 상관없고 열을 3열로 맞추라는 의미
    => (3,4) => (4,3)
  • reshape(-1,6) => (2,6)

cf. (3,3,64) 3x3이 64개 => (-1,8) 가능

  • tensor이면 높이, 행렬이면 열 기준으로 맞추라는 의미
