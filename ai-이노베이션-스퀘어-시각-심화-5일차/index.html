5일차

softmax 함수

출력층이 이항 노드이면 sigmoid를, 다항 노드이면 softmax를 쓴다.

전체 데이터 중에서 해당 데이터의 porsion.. 확률을 알기 위해 사용한다.

예제

딥러닝을 이용하여 강아지, 고양이, 코알라, 다람쥐를 분류할 때
각각의 이미지는 32 x 32 x 1 shape 을 가진다고 할 경우,

  1. 입력층(input layer) 노드 개수 32x32
  2. 출력층(output layer) 노드 개수 4 (activation function은 softmax)
  3. 학습을 마친 후, 임의의 1개 이미지에 대한 prediction 수행하였을 경우, prediction 에 대한 shape (1,4)
  4. ] 강아지 = 0, 고양이 = 1, 코알라 = 2, 다람쥐 = 3 으로 정답을 가정할 경우,

prediction 결과값이 [0.2 0.1 0.6 0.1] 인 경우, 입력된 이미지 2

insight

컴퓨터 비전 아키텍처의 내부를 들여다 볼 수 있는 창을 제공한다.
단순히 최대값만 뽑아 내는 거라면 sigmoid로도 충분하다.

ReduceLROnPlateau, Modelcheckpoint, Earlystopping ...

Model checkpoint

valid score가 좋아질 때마다 저장 => 가장 좋았을 때의 w, b를 찾는데 유용

Early stopping

val loss가 n epochs 이상 개선되지 않으면 조기 종료
=> hyper parameter의 일종 : 데이터에 따라 loss graph 양상이 다를 수 밖에 없어서...
=> Modelcheckpoint, Earlystopping ... 종종 이 둘을 같이 쓰기도 한다.

ReduceLR On Plateau

val loss가 n epochs 이상 개선되지 않으면 facter param으로 학습률 LR 조정
DL 경험이 많거나 데이터의 특성을 잘 아는 사람이 사용해야 한다

  • ReduceLROnPlateau, Earlystopping 는 Develop phase 개발 단계에서 쓰지말고, release phase에서 쓸 것을 권장한다.
예제

지난주에 실습했던 FashionMNIST, ReduceLROnPlateau/ModelCheckpint/ModelCheckpint,EarlyStopping 콜백을 구현하였다.

cf. paper에서 데이터 선택 자체가 bias가 포함된 행위이다.

결측치 nan
  1. Missing Data 개수 확인 => df.isnull().sum()
  2. 각 열(column)에 있는 각각의 Data 개수 확인 (NaN 제외한 데이터 개수)

df[‘Name’].value_counts(), df[‘Country’].value_counts()...
3) NaN 값이 있는 행(row) 모두 제거
4) Missing Data 를 특정 값으로 변경하기 (각 열의 NaN) : df[‘열이름’].fillna(변경값, inplace=True)
5) Missing Data 를 특정 값으로 변경하기 (모든 NaN ) : df.fillna(변경값, inplace=True)

cf. [appendix] mean(), median(), replace()
fillna() 에서 NaN 을 특정 값으로 변경할때 mean() 또는 median() 등으로 바꾸는 경우가 많음 (통계의 오류는 감안 해야함)

replace() 함수 이용하여 NaN ⇒ 특정값 또는 특정값 ⇒ NaN 으로 변경하는 경우도 있음
(특정값은 일반적으로 outlier 경우가 일반적임)
