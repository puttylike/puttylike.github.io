메모

cf. 부스트코스 - 자바스크립트의 시작
cf. 생활코딩 - WEB2 JavaScript

(1) JavaScript

  • 사용자와 동적으로 상호작용하는 WEB을 만들기 위한 언어.
  • HTML 위에서 동작하는 언어라서, script 태그 안에는 Javascript 코드를 사용하면 두 언어를 같이 쓸 수 있다.
  • Javascript로 쓴 코드는 동적이고, HTML은 정적이라는 차이가 있다.
    cf. html에 적힌 1+1은 1+1로, (script 태그 내) javascript에 적힌 document.write(1+1);은 2로 나온다.

(2) onclick 속성
HTML 태그 안에서 onclick 속성은 javascript 코드를 가지게 된다.
cf. button 클릭 시, onclick 안의 js를 실행 

<input type="button" value="hi" onclick="alert('hi')">

(3) 이벤트

  • 위와 같이 웹 브라우저에서 일어나는 유용한 사건.
  • 총 10~20가지 정도의 이벤트가 존재한다.
    cf. javascript keydown event 검색 => onkeydown 이벤트

  • onchange 이벤트
    입력창에 입력한 후 바깥쪽을 클릭했을 때를 기준으로 그 전과 내용이 바뀌었는지 확인 

    <input type="text"onchange="alert('changed')">

(4) 콘솔

  • 파일이 아닌 간단한 코드를 실행하고 싶을 때 사용.
  • 콘솔을 이용하면 파일을 만들지 않고도 바로 Javascript 코드 실행 가능.

  • 콘솔을 통해서 입력된 코드는 현재 페이지에서 즉석으로 실행된다.
    cf. 어떤 문자열(Hello, World!)의 길이를 알고 싶다면? 

    alert('Hello, World!'.length)
