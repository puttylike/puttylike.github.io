1. 의료정보학 개요

1.1. 의료정보학 개요

Biomedical informatcis = 의생명정보학, 생명의료정보학, 의료정보학
인간의 건강을 향상 시키기 위한 과학적 탐구, 생체학 데이터 정보 및 지식 사용을 연구하고 추구하는 다학제 분야

cf. 보건의료정보학
인간의 건강 향상 / 보건의료분야의 문제 해결 / 의사결정을 위한 데이터, 정보, 지식의 저장과 검색 및 활동을 다루는 과학 분야 / 응용과학을 쓰는 응용과학의 일부

의료정보학, 생명정보학 -> 보건의료정보학의 방법, 기법 이론 -> 현장 ) 생명정보학 / 영상정보학 / 임상정보학 / 공공보건정보학

그런데, Biomedical informatics 은 의료일까? 정보학일까?

Medicine (Medical Science), 이전에는 기본적인 구조는 정보과학 이라고 했다.

정보과학의 정보 흐름을 보면,
기본적인 지식 + 새로운 데이터 입력 => 추론 및 의사결정 => Action plan 수립

이는 환자를 치료하는 정보 흐름과 비슷하다.

의료정보학의 정보 흐름
기본적인 의학 지식 + 환자 정보 입력 => 진료 및 치료 의사과정 => 치료 실행

이래서 의료는 본잘적으로 정보학이다.

  • 정보 의학 : 정보 를 활용 해 진료 하고 의료 를 실현 하는 것

Information prescription / Intervention
정보를 처방 -> 앱 처방

= 정보의학의 개념을 단단하게 이야기 해주는 근거

보건의료 data, 보건의료 science, 컴퓨터 science (+) 역학, 의공학, 인지과학, 문하인류학, 경영학 => 의생명정보학

  • 의생명정보학 분류 by 미국의료정보학회

    • 중개생명정보학
    • 임상연구정보학
    • 임상정보학
    • 소비자건강정보학
    • 공공보건정보학

의료정보기술
Information Technology / Information Communication Technology를 건강 / 헬스케어에 접목

  • EMR
  • Computerized Provided order entry
  • 임상의사결정지원시스템
  • Consumer health informatics
  • Mobile computing
  • Telemedicine

Digital Health
= Digital technology, information technoloy, genome technology + 의료 => 보건의료계의 문제들을 RCT, RT를 활용해 해결하고자 함

대표적인 Information technology에는..

  • Clinical decision support
  • Health information system
  • Machine learning
  • Natural language processing

의료정보기술과 컴퓨터 역사는 초반에 일치한다.
=> 병원정보시스템이 출발하면서 의료정보기술 역사가 시작되었다.

cf. 1966년

  • GE, 메티넷 개발 및 운영
  • 중앙에 컴퓨터를 두고 단말기에 접속하여 컴퓨터를 활용
  • 병원 업무, 재정 업무에 사용할 목적으로 개발하였으나 제한점이 다수 발견

  • 의료정보학의 역사

    • 1949년, 독일에서 의료정보학과 관련 조직 개설
    • 1950년, 덴탈 프로젝트를 위한 Informatics program 개설
    • 1960년, MYCIN (expert system) 개발로 세계적 주목
    • 1965년, National Library of Medicine 서비스 제공
    • 1968년, 유타에서 세계 최초 Medical informatics department 설립
    • 1976년, 유럽의료정보협회 설립
    • 1979년, 국제의료협회 설립
    • 1987년, 대한의료정보학회 설립
    • 1988년, 미국의료정보협회 설립
    • 1999년, 아태의료정보학회 설립
  • 미래 의료를 끌어갈 Technological trend Top 10
    인공지능 / Virtual reality / Augmented reality / Healthcare Wearables / Genomics / Clinical trial simulation / 3D Printing / Nano technology / Medical Tricorder / Robotics in Healthcare
  • Clinical Medicine
    의사가 관찰 => 입원 후 병상 모니터링 => 의미 있는 데이터 선택 => 가설 및 통계적 분석 => 진료 현장 : 병원
  • 디지털 헬스
    자동적 기록 => 어디에서나 모니터링 => 모든 데이터 수집 => Machine learning / AI => 진료 현장 : 병원 + 병원 밖

한국 보건의료관련 비용 상승 : 질병 관련 부담 / 노령화 / 접근성 / 환자 안전 문제 => mobile, wearable 활용 필요

기존에는 의사 중심이었안, 환자 중심적 서비스가 요구딘다.

환자 중심적 서비스 : 보건의료 서비스 당자자인 환자를 포함학, 이때 등장한 것이 환자 참여, 환자 권한 부여

환자 측면의 진료 결과 => 환자 측면의 결과 평가 및 수가 반영

  • Patient reported outcome(PRO)
  • Patient centered outcome(PCO)
  • Patient experienced outcome(PEO)

  • 환자 참여 시스템 도입 효과

    • 진료 결과와 환자 만족도 향상
    • 보건의료전달 비용 감소
    • 치료 과정의 파트너십 형서으로 순응도 향상
    • 질병의 재발 감소
    • 부작용 발생 시 상대적으로 잘 받아들임
  • 환자 중심 서비스 도구
    기존에는 인터넷이 도구 => 최근에는 Mobile technology : 모바일 헬스 App 등장 / Patient portal, Personal health record 등장
  • 모바일 헬스
    개인적으로 건강 정보에 접근할 방법이 별로 없음
    => 모바일 폰 등장 : 건강 정보의 접근성 향상 / 자기 건강 정보 제공 도구 / 자신의 건강을 체크할 수 있는 도구

1.2. 의료정보학과 의료정보기술의 역할

보건의료체계에서 의료정보학과 의료정보기술의 역할 ?

Institution of Medicine(2001)
과거 보건의료체계에 많은 오류 발생 / 오류를 줄이기 위한 행동, 대책, 의료 질 향상 대책 강구 => Crossing the quality chasm 보고서 발표

  • Quality를 위해 필요한 성질

    • 안전
    • 효과성
    • 환자중심성
    • 시기적절성
    • 효율성
    • 공평성

=> 지식에 근거한 진료 (sum 데이터 제공 / 처방 세트 / 규칙에 따른 Alert ) + 기술을 적극적으로 활용 (임상정보시스템과 어플리케이션 협력)

환자 안전 향상 방법 (from NEJM 2003;348:2526-34)

  • 의사소통을 향상
  • 지식을 신속하게 접근할 수 있게 함
  • 다양한 정보를 한눈에 볼수 있게 함
  • 계산에 도움
  • 의사결정을 도움

EMR 환자 안전 향상 방법

  • 환자 내원 시 기록에 도움
  • 의료진 간 의사소통 향상
  • 환자 정보 요약
  • 접근성 향상
  • 처방 실수 예방으로 환자 안전 향상
  • 연구 및 Quality improvement에 활용

정보가 너무 많은 문제 발생 = information tsunami

NLM에서 매주 8,000개 Articla 발간 (2000년) = 수치로 계산하면 매일 20개 이상 365일 Article을 읽어야 함
=> 스스로에게 필요한 정보를 활용하고 접근해야 함

  • Biomedical sciecne -> Medical practice
    -> 이렇게 옮겨가는데 17년이 걸리고, 14%만 남는다.
    (from Edward H. Shortliffe, James J. Cimino. Medical informatics: computer applications in health care and biomedicine. Sorinaer Science & business Media: 2006 Nov 11.)

=> 의생명정보학적 기법을 활용해, Research 에서 Practice 까지의 과정을 축소하려는 것 : 의생명정보학

임상 의사결정의 문제점

  • 정보가 없는 상태에서 진행
  • 데이터가 너무 많아 결정에 어려움
  • 접근이 힘든 상황에서의 의사결정
  • Bias의 개입

30개 정도의 bias가 관여헸고 지금은 더 많다고...
(from The Importance of Cognitive Errors in Diagnosis and Strategies to Minimize Them. Acd. Med. 2003;78:775-780)
