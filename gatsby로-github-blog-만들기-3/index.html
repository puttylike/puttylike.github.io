Github Blog 만들기3 - 수식, 코드 스타일 추가 (by gatsby)

한글날이 있길래, 월요일 연차를 붙여 썼다. AI 이노베이션 강의 내용 복습하고 시간이 남아서 블로그를 업데이트 해보았다.

어느 덧 블로그 시작한지 1달이 다 됐다. 깃허브 블로그 쓰다 보니까 수학 수식이나 프로그래밍 코드가 안 이쁘게 보여서 (...) 구글링을 통해 보완책을 마련해 봤다.

이 두 개의 plugin 모두 gatsby-transformer-remark 하위에 추가하면 된다.

메모

  • math > https://www.gatsbyjs.com/plugins/gatsby-remark-mathjax/
  • code > https://www.gatsbyjs.com/plugins/gatsby-remark-prismjs/
  • prismjs thema > https://github.com/PrismJS/prism/tree/1d5047df37aacc900f8270b1c6215028f6988eb1/themes

깃허브로 블로그 만들기 - 3

(1) mathjax 적용 방법 (with Helmet)

  1. gatsby-remark-mathjax 설치
$ npm install --save gatsby-transformer-remark gatsby-remark-mathjax
  1. gatsby-config.js 파일 수정

기존의 gatsby-transformer-remark 에 추가하지 않고, 새로 만들었다.

module.exports = {
  plugins: [
    {
      resolve: `gatsby-transformer-remark`,
      options: {
        plugins: [
          `gatsby-remark-mathjax`,
        ],
      },
    },
  ],
}
  1. blog post 관련 template 수정 (with Helmet)

내 경우 /src/templates/blog-post.js 를 사용 중이어서 이 파일을 수정했다.
script 추가할 방법을 찾다가 Helmet 안에 그냥 집어 넣었다.

$ npm install helmet --save
// 위 쪽에 추가하고
import { Helmet } from "react-helmet"

// 이 아래는 return 내 본문 쪽 부분에 추가한다.

<Helmet>
  <script>{"MathJax = {tex: {inlineMath: [['$', '$']]}}"}</script>
  <script
      id="MathJax-script"
      async
      src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js"
  ></script>
</Helmet>

내 경우 post 부분이 이래서 </Box> 위에 script를 이 안에 집어 넣었다.

<Box
  dangerouslySetInnerHTML={{ __html: post.html }}
  css={{ lineHeight: '30px', position: 'static' }}
/>
</Box>

위 코드를 추가하면, (블로그 게시물) 마크다운 파일에 적용된다.

  1. gatsby-browser.js 에 추가

src 디렉토리 상위 경로에 gatsby-browser.js 파일을 생성하고, 아래 내용을 입력한다. (내 경우 puttylike.github.io 디렉토리였다.)

require("prismjs/themes/prism-okaidia.css");

여기까지하면 기본적인 세팅이 완료된다.

  1. 기타 option

위의 prism-okaidia.css 말고 다른 파일을 쓰고 싶다면, 여기 링크에 있는 다른 css 파일을 쓸 수도 있고, 직접 css 파일을 만들어 import하는 경우도 있겠다.

나머지 내용은 공식 문서 링크를 참고하면 될 듯 싶다.

위 링크에 있는 내용이다만, 소스 코드 옆에 line numbering을 하고 싶다면, gatsby-browser.js 에 아래 내용을 추가하면 된다.

require("prismjs/plugins/line-numbers/prism-line-numbers.css") // Add line numbering

(3) 수식 & 코드 작성 방법

  1. 수학 수식 작성하기

수식 앞뒤로 $$ 또는 $ 를 감싸면 된다.

$$f(x) = 1/(1+e^x)$$  
$g(x) = x^3+2x+c$

$$f(x) = 1/(1+e^x)$$
$g(x) = x^3+2x+c$

한번씩 로딩이 뭔가 제때 안 되는 느낌적인 느낌(?)이 드는데, 일단 되니까 이대로 마무리지었다.

  1. 코드 작성하기

```javascript 과 ``` 사이에 소스 코드를 넣으면 된다.

plugins: [
  {
    resolve: `gatsby-transformer-remark`,
    options: {
      plugins: [
        // 20201012 programming code
        `gatsby-remark-mathjax`,
        // 20201012 mathematical expression
        `gatsby-remark-prismjs`,
        },
      ],
    },
  },
],
