1차시) 금융혁명 핀테크의 바람이 불고있다!

1. 핀테크란 무엇인가?

  • 핀테크 = 금융 기술 = Financial + Technology

=> 금융과 IT의 융합을 통한 금융서비스 및 산업의 변화를 통칭.

옛날에는 은행 등 직접 가야만 할 수 있었는데, 인터넷의 발달로
인터넷만 연결돼 있다면 언제 어디서나 업무 처리가 가능해 졌다.

컴퓨터가 있어야 했고, 공인인증서와 회사 별 보안프로그램이 필요했지만,
조금은 어렵고 불편해도 금융 서비스는 보안 때문에 그런 거라고 여겨졌다.

2. 디지털 환경 변화의 3가지 영향

잔잔했던 금융에 3가지 혁명의 바람이 불어오며 바뀌었다.

  1. 스마트폰
    • 핸드폰은 단순 전화기에서 스마트한 모든 것을 할 수 있는 기기로 진화했다.
    => 하나은행은 최초의 모바일 아이폰 어플인 하나N뱅크 출시 : 모바일 어플 출시

이로 혁명의 바람이 시작됐다. 프로그램 설치는 필요 없어졌으나,
서류 제출 요구/이용 시간 제한이라는 불편한 점은 여전했다.

(왜 은행에 가야 할까? 통장 재발급 비용은 왜 필요할까? 영업 시간은 왜 짧을까? 대출 서류는 왜 이렇게 많을까?)
고객들은 이전과 달리, 서비스에 대한 기대를 일반 회사 수준으로 금융회사에 요구하기 시작했다.

  1. 온라인 전문은행

    • 카카오뱅크, 케이뱅크 등 인터넷 전문은행 등장 (무점포)

    : 점포 = 스마트폰
    ex) 비대면 계좌 개설
    ex) 금융의 상품화 - 무조건성 고금리 상품 / P2P 상품 카톡 전송

  2. 빅테크

    • 삼성페이 from 삼성전자(테크기업)

    => 기술이 금융을 이끌게 되었다. : 핀테크 -> 테크핀
    테크 기업이 편리한 결제 시스템을 만들어도 결국 금융 기업과 손 잡아야 서비스가 가능하니, 공생간계 혹은 기 금융사에 유리했다.
    => 금융을 모르는 테크 회사들이 금융 서비스를 새로 공부해서 만드는 것보다, 금융 기업들이 테크 인력을 활용해 서비스를 만드는 일이 더 쉽다.

    • 금융 서비스 재해석 : 토스 사용자가 늘며 국내 은행들과 제휴를 하게 됐다. 이후, 은행 대부분이 간편송급 서비스를 하게 됐다. 그런데, 토스는 간편송금을 넘어 금융 서비스를 재해석하기 시작했다.

    ex) 온라인 - 카카오뱅크(21주 적금), 토스(증권사 설립 추진 등), 네이버(파이낸셜 설립 후 CMA 통장 개설로 고객 유치)
    ex) 오프라인 - QR결제 / 모바일 결제

IT의 거대한 공룡들 테크 기업이 금융 시장에 영향력을 높히고 있는데, 이를 빅테크라 부른다.
ex) Applepay, Alipay, wechatpay, googlepay ...

3. 핀테크 집중 분야

앞으로 핀테크에서 어느 부분에 집중해야 할까?

  1. 데이터 : 다양한 데이터를 수집하여 초맞춤화된 금융서비스 제공

돈은 끊임없이 순환한다. (이동->먹고->사고->저축->투자->...)
* 일상 플랫폼 (카카오) +금융 => 데이터 => 초맞춤화, 초개인화 서비스 가능 : 빅데이터 * 데이터를 분석해서 유의미한 결과 도출 : 인공지능

  1. 보안 : 블록체인 등 개인정보보호를 위한 보안기술의 발전

    • 공인인증서->보안카드->OTP, 이게 스마트폰 하나로 대체되면서 보안은 중요한 화두가 되었다.

    • 더욱 커지고 있는 기업의 보안 책임

    => 분산장부, 블록체인 방식 등 보안과 관련된 기술은 발전 중이다.

이런 이유에서 핀테크는 이미 무한경쟁의 시대에 접어 들었다.

...핀테크의 핵심에서 어떻게 변화되고 있을까?

2차시) 변화하는 금융 비즈니스 모델

1. 금융 서비스 모델

언제부터 금융 서비스를 이용하기 시작했을까? 옛날 수렵생활에서 정착생활을 하게 되면서, 사유재산이 생기기 시작했다.
집에 돈을 보관하는 게 어려워지며 은행이 생겼다.
은행 = 안전한 보관 => 돈을 맡기면 이자를 주는(?) 이상한 시스템.

은행은 고객이 예치한 예금에 이자를 제공하는 한편, 고객에게 대출을 통해 원금과 이자를 받는다.
=> 예금한 돈과 대출한 돈의 차이 : 예대마진
돈이 늘다 보면, 투자를 하거나 보험을 만들어 운영할 수 도 있게 되었다.

  • 은행 본연의 역할 => 돈이 중심 => 다양한 금융업으로 확장

  • 금융업을 하고 있는 회사들은?

제공하는 금융서비스의 성격에 따라 분류하면,
은행(일반은행, 특수은(상호저축은행, 신용협동기구, 우체국예금, 종합금융회()/비은행예금취급기관/보험회사/금융투자회사(투자매매중개업자, 집합투자업자, 투자일임자문업자, 신탁회사 등)/기타금융기관/금융보조기관

cf. 메가뱅크 : 은행을 합병한 큰 투자은행

복잡한 금융서비스들을 한 곳에서 처리하게 된다면?
투자를 전문으로 하는 투자 은행을 만든다면?

...그러다 2008년, 리먼 브라더스 사태 = 은행의 부실이 알려진 사건

cf. 뱅크런(BankRun)

금융위기 이후 정부차원의 긴급지원금 투입했으나, 임원 보너스 등에 쓰인 사실이 알려지며, 월가 점령하라는 대규모 시위가 일어났다. => 기존 금융회사에 대한 실망감으로 작고 빠른 테크 기반의 금융회사가 생겨나기 시작했다. 핀테크가 폭발적으로 성장할 수 있는 기폭제를 마련된 셈이다.

3. 은행, 증권, 보험업의 변화

금융위기 이후 침체된 금융을 살린 건 Digital의 영향이 크다.
모바일 혁명이 일어나며 관련 산업이 성장하기 시작했다.

toss 송금 / transferWise 해외 송금 / Kickstarter 투자
=> 많은 영역 중 1가지를 해서 성공했다.

작고 빠른 테크 기업이 생겨나며 눈에 보이는 비교할 수 있는 서비스로 바뀌며 경쟁이 심화됐다.

더 나아가 송금 따로, 대출 따로, 환전 따로.
다양한 곳을 이용하게 됐다. = 언번들링 (Unbundling) 현상

=> 기존 금융업에는 심각한 위기이자 도전으로 자리 잡게 디었다.

따라서 금융업에 핀테크가 적용된다는 건,
금융 회사에서 하는 수많은 서비스를 고객 입장에서 더욱 편리한 디지털 금융생활을 하고, 업무를 하는 입장에서는 디지털 트랜스포메이션으로 이어지게 되었다.

  • 디지털 트랜스포메이션 = 디지털 혁신

: A to Z 고객 서비스의 디지털화 with 인공지능/빅데이터/모바일/초맞춤화 개인화

cf. 혁신 = 고객의 입장에서 편의성에 맞춰 변화됨

  • 은행업의 가장 큰 변화 = MOBILE ONLY

금융사별 웹사이트의 제한적 서비스
=> 통장 개설 / 투자 / 대출서류 제출 / 아파트 담보대출 with 스마트폰
이를 모바일만으로도 모든 것이 가능!

cf. 2019년 말 오픈뱅킹 서비스 시행으로, 주거래도 더 쉽게 바뀌게 되었다.

3차시) 금융 빅데이터의 이해

1. 빅데이터란?

cf. 황금시간대
과거 광고계) 오후 9시 이전(뉴스), 오후 10시 이전(드라마)
=> 요즘) 오후 8시(저녁 시간 후), 오후 11시(하루를 마감하는 시간)

모든 활동에 관여하는 스마트폰.
이 스마트폰을 사용하며 끊임없이 데이터를 만들어내게 된다.

  • 빅데이터 : 모두와 모두가 만들어내는 막대한 양의 데이터

=> 분석 기술의 발달로 유의미한 결과 도출 가능

2. 금융권에서의 빅데이터 활용

금융에서 가장 필요로 하는 정보는? 결제 데이터
=> 고객의 결제 데이터를 활용한 맞춤형 금융 서비스 제공

활용하고 있는 고객의 데이터는 고객에게서 오게 된 거다.
그 데이터가 한 기관 내에서만 데이터가 활용된다면, 서비스의 경쟁력이 약해질 수 밖에 없고 확장성에 한계가 있을 수 밖에 없다.
=> 데이터 3법(개인정보보호법/신용정보법/정보통신망법) 이 개정됐다.

기업은 데이터의 중요성에 더해 데이터의 보호에도 신경을 쓰게 됐다. 데이터를 다루는 분야는 더 성장할 수 밖에.

금융권에서는 이런 빅데이터를 어떻게 사용하고 있을까?

cf. 신한카드 - 금융권 최초 초개인화 서비스 런칭 (2019.05) = AI 알고리즘으로 고객의 TPO(시간/장소/상황) 예측해 맞춤형 혜택 제공

  • 카드사의 빅데이터 활용 사례

=> 비트코인과 블록체인의 단점을 보완한 기술 등장
= 이더리움 (2세대 블록체인) 탄생 :
블록의 크기 무제한 / 거래속도 20초 이내 / 강제 계약 설정 (Smart Contract)

ex) 집 구매 절차 : 판매자 - 구매자 >> 공인중개사/법무사/행정처리 시 소유권 성립
=> 구매자가 판매자에게 입금 시 소유권 성립한다면?

  • Smart Contract : 개인-개인 다이렉트 연결하는 강제 계약 = 탈중앙화

cf. 이외에도 가상하폐는 많다. (다만, 비트코인과 이더리움이 만들어진 목적과 구동되는 방식이 유사하다.)
