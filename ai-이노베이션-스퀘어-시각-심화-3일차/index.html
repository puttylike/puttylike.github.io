그래서 직접 타이핑 해보고 있다. 작년에 기초(중급) 과정과 다른 점이라면, 파이썬 기본적인 내용은 생략되었고, 문제에 대해서 고민할 시간이 조금 촉박해진 점 정도이다.

3일차

신경망

시스템은 input, output 및 알고리즘에 대한 정의가 있어야 한다.
머신러닝, 딥러닝 시스템도 마찬가지다.

X * W + b
(X에 W로 평균적인 변화를 줘서 b로 조정)

  • 딥러닝의 궁극적인 목표 : 정답 T 와 계산값(output) Y의 차이인 loss 손실값를 최소화되는 W, b를 찾는 것

cf. a loss/a w = 현재 w 값이 조금 변할 때 loss는 얼마나 변하는 가
=> 미분은 현재 값이 있어야 의미가 있어서 w, b를 랜덤으로 초기화

  • 손실 함수
  • 모든 손실함수에서 나타나는 공통점 2가지
  1. 입력 값 개수만큼 합치고(시그마) 나누어 평균을 내고 있습니다.
  2. Y - T
  • 수학적 insight

정답과 계산값 차이의 평균

은닉층이 있으면 딥러닝 아키텍처이다.

feed forward
입력값 data를 통해 출력값을 만들어내는 과정

입력층 node는 정답을 만들어내는 값 갯수만큼, 출력층 node는 정답의 갯수만큼 필요하다.

  • 활성화 함수

변곡점을 갖고 있는 함수로, 입력 데이터를 특정 범위 이내로 바꾸는 역할을 한다.
그래서 활성화 함수가 되려면 변곡점이 필요하다.

sigmoid는 가장 basic하나 성능이 낮고, relu는 image에 잘 쓰이고, tanh는 주로 LSTM, RNN에 쓰인다. 다만 이들은 변곡점이 하나라 binary 분류만 가능하다.
=> 2개 이상의 데이터를 분류하기 위해 one-hot cneoding을 활용한다.

  • one-hot encoding

2개 이상의 정답이 있는 경우, 그 정답을 표현하는 방식

  • 딥러닝 순서
  1. data definition => numpy
  2. init w, b => 모델 구축 과정
  3. feed forward, loss function => 모델 compile 과정
  4. learning => model.fit

TF 2.0

  • 딥러닝 순서 in TF
  1. data definition => numpy (tensor로 만드는 작업)
  2. model create : init w, b => 모델 구축 과정 : sequential/functional
  3. model compile : loss, learning(optimizer, w, b) ... cf. SGD, ADAM
  4. learning (model.fit) : for문 2개로 볼 수 있다

사용 친화적이나 성능이 낮아졌다.
TF가 forks가 많다.

  • keras vs tf vs pytorch

개발자 관점에서 largest dataset 때문에 tf가 좋고,
실무 관점에서 TDD 방법으론 rapid prototyping 때문에 keras가 좋다.
=> 그래서 이 수업에서는 tf로 배운다.

  • keras in tf 2.0

keras의 창시자 프랑수와 슐레는 tf.keras를 권장한다.
keras의 모델은 신경망 자체이고, 모든 모델의 기본 단위는 layer로 나타낸다.

  • TF에서 model 생성 순서
  • sequential : model 생성 => layer ADD
  • functional : layer 생성 => model (input, output) 생성

cf. ANN에서 신경망 layer는 1차원 벡터이다.

  • loss

mse는 정답이 실수로 나온다.

TF 기본 아키텍처는 모델이다. Sequential, Functional,
그리고 Subclassing이 있는데 Subclassing은 안 쓴다.

import tensorflow as tf

from tensorflow.keras.models import Sequential, Model
from tensorflow.keras.layers import Flatten, Dense, Input,concatenate
from tensorflow.keras.optimizers import SGD

import numpy as np

input_ = Input(shape=(4,))

layer_ = Dense(8, activation='relu')(input_)
layer2_ = Dense(16, activation='relu')(layer_)
output_1 = Model(inputs=input_, outputs=layer2_)

input2_ = Input(shape=(8,))
layer3_ = Dense(8, activation='relu')(input2_)
output_2 = Model(inputs=input2_, outputs=layer3_)

layer4_ = concatenate([output_1.output,output_2.output])
output_ = Dense(10, activation='softmax')(layer4_)

model = Model(inputs=[input_,input2_], outputs=output_)
model.summary()
