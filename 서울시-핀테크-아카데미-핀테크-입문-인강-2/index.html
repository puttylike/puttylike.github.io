6차시) 스테이블 코인과 CBDC의 이해

12/27 이번 강의는 좀 생소한 내용이었다. 리브라는 알긴 한데...

1. 코인의 발행

스테이블 코인 Stable coin ≒ 가상화폐 / CBCD

  • 코인 = 누구나 발행 가능한 화폐 / 거래하는 방식만을 제공

코인의 목적 = 코인으로 무엇을 할 수 있는가?
=> ex) 기업에서 발행 시, 코인을 발행한 기업의 서비스만 이용 가능
=> 비트코인의 경우, 채굴 방식이 있는데, 이 방식에서는 거래하는 방식만 제공
=> 그래서 코인을 구매하려고 하는 회사에 대한 믿음이 필요
=> ICO를 통해 공개하여 가상화폐 거래소에 상장한다.

코인의 문제점

  • 가격 측면 - 큰 가격 변동성
    cf. 주식 시장과 달리 가상화폐 거래소는 24시간 거래가 가능하다.
  • 해킹 측면 - 시스템 문제
    => 블록체인 base의 코인과 달리, 해킹에 노출된 거래소 : 인적 리스크 (내부자 도덕적 해이) / 외부 공격

2. 스테이블 코인의 개념

지금까지 가격 변동성 / 탈중앙성을 언급했는데, 아직까지 코인은 보안에 취약하다.

=> 스테이블 코인 탄생
: 기존의 화폐 또는 실물자산과 연동시켜 가격 안정성(달러, 원)을 보장하는 화폐로,
법정화폐 담보형(Fiat-Collateralized Stablecoin), 암호자산 담보형(Crypto-Collateralized Stablecoin) / 무담보형(Non-Collateralized Stableoin) 으로 분류된다.

  • 스테이블 코인이 가격을 고정시킬 수 있는 방법
    법정 화폐 담보형 - 특정 기관에 달러 등을 쌓아 두고, 그만큼의 코인을 발행
    ex) GEMINI - Gemini Dollar (GUSD 제미니 달러) / Tether (Tether USDT 테더) / TrueUSD (TrueUSD TUSD)
    cf. 이외에도 암호화폐를 담ㅂ로 만드는 방식, 알고리즘으로 만드는 스테이블 코인이 있다만, 법정 화폐 기반이 가장 안정성이 있다 볼 수 있다.

100% 신뢰할 수 있는가?
블록체인 방식이라 보안 면에서 해킹 대응은 용이하나,
(1코인에 해당하는 1달러가 예금돼 있는지) 장부미공개라 화폐가치 관점에서 불안정해 보이긴 하다.

현금과 스테이블 코인의 비율이 같다면, 굳이 스테이블 코인으로 갖고 있을 필요가 있을까?
ex) 네이버페이 : (코인은 아니지만) 충전 포인트 결제 (3% 적립)

  • <디지털화폐 : 스테이블 코인의 부상>
    국제통화기구 said 지불수단 혁신 네트워크 효과, 디지털 라이프와 사용자 중심 설계 => 정부나 금융적 입장은 우호적!

3. CBDC와 페이스북 리브라

스테이블 코인은 왜 주목받았는가? 페이스북 때문.

  • 페이스북 : 2019.06 전세계 금융 소외계층 17억명에게 금융서비스를 제공하겠다며, 페이스북 암호화폐 리브라 개발

cf. 세계 SNS : 왓츠앱 / 페북 / 인스타 순...
=> 페이스북 월드 외 Visa / Ebay / Paypal 에서도 쓸 수 있게...
=> 전세계를 자극한 페이스북 리브라 프로젝트
=> 뱅크런 등의 우려로 리브라에 대해 미국의 규제
=> Visa / Ebay / Paypal 탈퇴 후 Libra White Paper 리브라 백서 수정
: 지역기반 단일 화폐 이용 방식으로 변경 = 단일 통화 코인에서 다중으로 변화 (2020년말 오픈 예정)

  • 2020년 12월, 최근 디엠으로 개명했고, 빠르면 2021년 1월 출시라고 한다.

중국 : 위챗 > 위챗페이 / 알리페이
= 현금 결제에서 빠르게 모바일 결제로 전환

  • CBCD (Central Bank Digital Currency) : 중앙은행에서 발행하는 디지털 화폐로 국가가 통제하는 신용화폐와 같은 구조.

* 페이스북 리브라 : 페이스북 암호화폐로 다국가에서 사용할 수 있는 통화를 만들고자 했으나, 미국의 규제로 중단

  • 페북&중국이 당긴 디지털 화폐 전쟁
    2016년 중국, 디지털화폐 연구소 설립 - 디지털화폐에 대한 불신 해소
    => CBDC 베이징 동계올림픽 즈음 2022년 상용화 계획
  • CBDC의 발행 이유
    지폐발급 비용의 절감
    관리와 통제
    화폐 주도권 탈환

ex) 중국 - 2020년도 10월, 디지털화폐 200위안 총 5만명 에게 제공하여(=> 17억), 7일간 소비 실험 : 시민들 새로운 결제방식 거부감 없음

  • 이미 전세계는 CBDC 준비 중 - 다른 나라에 대비한 화폐 주권 보호도 주 목적
    => 한국은행도 마찬가지이다.
  • 한국 : 2020년 3월 ~ 12월 CBDC 컨설팅
    2021년 이후 파일럿 시스템 구축 및 테스트

7차시) 대출

1. P2P 대출 서비스

  • P2P 대출 서비스 = 불특정 다수로부터 투자금을 모아 대출을 원하는 사람에게 약속한 기간동안 이자를 받는 대출 서비스로 일종의 크라우드 펀딩 개념
  • 부동산 담보 대출 > 은행마다 금리 다름 서류 제출 후 확정 시 대출 실행되는 구조

급한 돈 대출을 개인에게 대출한다면?
(대부업 고금리 / 개인 사채)
=> 디지털 사채 = P2P 금융 : 개인간의 이동

  • 2005년 영국회사 ZOPA, P2P 대출의 시작
  • 신용등급에 따른 이자율 평가 > 위험등급일 수록 높은 이자 > 상환 리스크 최소화
    cf. 현재까지도 다른 P2P 회사들도 이 회사 방식과 유사
  • 개인들의 투자 - 소액 투자 가능 상품 당 10파운드 투자 > (그 이상은) 분산투자 방식 > 투자자의 리스크 ↓
  • P2P 리스크 - 채무 불이행
    => 세이프 가드 - 보험 도입 : 투자 손실이 발생했을 경우 미리 모아두었던 기금에서 보상
  • 특이하게, 2014년, 영국 금융업무 감독청에 규제 요청
    => 신뢰 구축 목적

1. 크라우드 펀딩의 정의

아이디어를 현실화하기 위해서는? 사람과 돈이 필요!
물건을 만들자 마자 팔 수 있다면? 미리 돈을 받고 그 돈으로 물건을 만들 수 있다면? = 크라우드펀딩

크라우드 펀딩 : 대중들이 모여 돈을 투자

  1. 리워드 펀딩 : 투자의 대가로 물건이나 서비스를 받는 형태
  2. 기부형 펀딩 : 물건이나 서비스를 제공받는 일 없이 돈을 기부하는 형태
  3. 대출형 P2P 펀딩 : 투자의 대가로 이자를 받는 형태
  4. 증권형 펀딩 : 투자의 대가로 지분을 받는 형태
  5. 리워드 펀딩 ex) 해외 - 킥트타터, 인디고고 / 국내 - 와디즈, 텀블벅

    => 성공 : 해당 기간 내에 100% 이상의 자금 모집 / 실패 : 해당 기간 내에 100% 이상의 자금 모집 X

  3. 이미 팔고 있는 제품인 경우 ex) 중국산 제품을 이름만 바꿔서 신규 제품인 것처럼 판매 / 국내에서 정식으로 판매되는 제품을 해외에서 저렴한 가격으로 가져와 판매
